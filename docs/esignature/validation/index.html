PDF Validation API Calls

POST - Validate a PDF Document​

Request​

POST @ /api/v1/validation/validateDocument

Request Body​

  • bytes : The PDF to be validated should be encoded in base64 format
  • name: Optional name for the pdf document
{
  "signedDocument" : {
    "bytes" : "JVBERi... ",
    "name": "important.pdf"
  }
}
